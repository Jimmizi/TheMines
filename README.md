# The Mines
Made for Ludum Dare 49 (Theme: Unstable)

Escape the mines before you run out of oxygen, but beware of the ghosts!

WASD - Movement

Left Click - Dig, Refuel

LCTRL/L - Toggle Lantern

Tip: Use the lantern on the ghosts.
